A new Flutter project.

## Getting Started

This project serves as a starting point for the redesign of the MO Marketplace.

A few resources related to this app:

- [App APK FILE](https://drive.google.com/file/d/1HIgcpVOL6z-OHXFXda1pXP5l1y11AeFc/view?usp=drive_link)
- [Sample Video](https://drive.google.com/file/d/171z0tdR9uxjmtTSFnveBNB2o4Xo2Gg8D/view?usp=drive_link)

## Features

- Home Screen: Banner, categories and other main products.
- Bottom navigation menu - home, categories, add, chat and login.
- Category List: Display the Categories.
- Products: Display products of selected Category.
- Product Details - Display name, price, description and other information
- Login and Register

## Tech Stack

- Frontend: Flutter.
- API call - [Dio] (https://pub.dev/packages/dio).
- State Management - [Riverpod] (https://pub.dev/packages/flutter_riverpod).
- Lazy Loading - [CachedNetworkImage] (https://pub.dev/packages/cached_network_image).

## Installation

- Step 01 : Clone the repository using the following command.<br>
git clone https://github.com/Jawhar27/mo_marketplace_test.git

- Step 02 : Run this command after opening the project.<br>
flutter pub get

- Step 03 : To Run the project.<br>
flutter run
